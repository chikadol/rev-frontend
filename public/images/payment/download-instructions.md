# 결제 버튼 이미지 다운로드 안내

각 결제 서비스의 공식 결제 버튼 이미지를 다운로드하여 이 폴더에 저장하세요.

## 1. 네이버페이
- 개발자 센터: https://developers.pay.naver.com/design/brand/button
- 파일명: `naverpay-button.png`
- 저장 위치: `public/images/payment/naverpay-button.png`

## 2. 토스페이먼츠
- 개발자 센터: https://docs.tosspayments.com/guides/design-guide
- 파일명: `toss-button.png`
- 저장 위치: `public/images/payment/toss-button.png`

## 3. 카카오페이
- 개발자 센터: https://developers.kakao.com/docs/latest/ko/kakaopay/common
- 파일명: `kakaopay-button.png`
- 저장 위치: `public/images/payment/kakaopay-button.png`

## 빠른 다운로드 방법
각 개발자 센터 페이지에서 제공하는 버튼 이미지를 우클릭하여 "이미지 저장"을 선택하거나,
브라우저 개발자 도구(F12)를 열어 Network 탭에서 이미지 파일을 찾아 다운로드하세요.
